## Descripción
Proyecto desarrollado para la asignatura Ingeniería de Software III con el propósito de aprender y practicar el uso de Git y GitHub. En este proyecto se aplican conceptos de control de versiones, gestión de cambios, trabajo colaborativo mediante ramas, commits y pull requests.
